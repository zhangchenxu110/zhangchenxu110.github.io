创建并删除用户

创建新用户

sudo adduser nutao

输入两次密码，后面的信息选择性填写

添加root sudo权限

sudo vim /etc/sudoers

删除用户
sudo userdel -r 【username】

添加 -r 会将用户目录也删除

开启root用户

# 开启root用户
sudo passwd root

允许桌面 root 登陆

sudo vi /usr/share/lightdm/lightdm.conf.d/50-ubuntu.conf

添加如下指令

greeter-show-manual-login=true

重启之后，就可以使用 root 用户登陆（需要手动输入用户名和密码）
